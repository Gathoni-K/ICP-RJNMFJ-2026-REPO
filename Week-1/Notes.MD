# Creating an App that prints Hello World.

## App setup.

-Initialize node project using:

    npm init -y

-Install necessary dependencies:

    npm install express cors dotenv
    npm install -D typescript @types/node @types/express ts-node nodemon

-Create the 'index.js' file at the root of our directory.

-This files acts as an entry point.

-We run our app using:

    node index.js

-Open the browser and input the port set as below:

    http://localhost:portNumber
