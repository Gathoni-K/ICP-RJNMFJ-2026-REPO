# Week 1: Environment Setup & Hello World

## Objective
Set up the backend development environment and implement a basic proof-of-concept application.

## Tech Stack
- Node.js
- Express

## Steps Completed
1. Initialized project with `npm init -y`
2. Installed Express: `npm install express`
3. Created `index.js` as the entry point
4. Implemented a basic route to return "Hello World"
5. Verified the server runs on `localhost:3000`

## How to Run
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `node index.js`
4. Open `http://localhost:3000` in your browser to see "Hello World"

## Notes
- This week focused on setting up a working backend environment.
- No frontend or database integration was included at this stage.
